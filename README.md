# Kumon Fishermall — Learning Center Website

A clean, responsive landing page for **Kumon Fishermall**, a locally operated Kumon center located inside Fisher Mall along Quezon Avenue, Quezon City, Metro Manila.

---

## 🌐 Live Preview

> Open `index.html` directly in your browser — no build step required.

---

## 📁 Project Structure

```
Kumon2.0/
├── index.html      # Main HTML structure & page sections
├── styles.css      # All styles, design tokens & responsive rules
├── script.js       # Mobile nav, scroll reveal & form handling
└── LICENSE         # MIT License
```

---

## ✨ Features

- **Responsive design** — mobile-friendly layout with a hamburger nav menu
- **Smooth scroll reveal** — sections animate into view as you scroll
- **Hero section** — full-viewport dark gradient with headline and CTA buttons
- **Stats bar** — Kumon global figures (60+ years, 50+ countries, 4M+ students)
- **About section** — center overview, learning philosophy, and why-choose-us cards
- **Programs section** — Mathematics, Reading & English, and Early Learners with level progression chart
- **Kumon Method section** — method pillars and founder quote block
- **Kumon Connect section** — digital vs. paper comparison with step-by-step flow
- **Location & Contact section** — address details and enrollment inquiry form
- **CTA banner & footer** — call-to-action with full footer navigation

---

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| HTML5 | Page structure and semantic markup |
| CSS3 | Custom design tokens, grid/flex layouts, animations |
| Vanilla JavaScript | Nav toggle, IntersectionObserver reveal, form handling |
| Google Fonts | Cormorant Garamond (serif headings) + DM Sans (body) |

No frameworks, no build tools, no dependencies — just open and run.

## 🎨 Design Tokens

Defined as CSS custom properties in `styles.css`:

| Token | Value | Usage |
|---|---|---|
| `--navy` | `#1A3A5C` | Primary dark background, headings |
| `--slate` | `#1E4976` | Hero gradient mid |
| `--steel` | `#2B6189` | Accent text, italic emphasis |
| `--sky` | `#3A8FC4` | Buttons, labels, icons |
| `--mist` | `#7FB3D3` | Subtle text, hero subtitle |
| `--silver` | `#C2D8E8` | Borders, dividers |
| `--cream` | `#EEF5FA` | Section backgrounds, card fills |
| `--white` | `#F7FAFD` | Page background |

## 🚀 Getting Started

No installation required.

1. Clone the repository:
   ```bash
   git clone https://github.com/EzekielLaguiab/Kumon2.0.git
   ```
2. Open `index.html` in your browser:
   ```bash
   # On Windows
   start index.html

   # On macOS
   open index.html
   ```

Or simply drag and drop `index.html` into any browser window.
